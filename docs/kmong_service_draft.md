# 크몽 서비스 등록 초안

## 서비스 제목

**사장님의 진짜 이야기를 4가지 콘텐츠로 생성해 드립니다**

---

## 서비스 한줄 소개

사진 보내고 5분 인터뷰하면, 네이버 블로그 · 인스타 릴스 · 캐러셀 · 쓰레드까지 한번에 만들어 드립니다.

---

## 서비스 설명

### 이런 분들을 위한 서비스예요

- 블로그 글 쓸 시간이 없는 사장님
- 릴스/숏폼 영상 만들고 싶은데 어떻게 시작해야 할지 모르는 분
- 인스타그램에 올릴 콘텐츠가 매번 고민인 분
- AI로 만든 티 나는 콘텐츠가 아닌, **내 가게의 진짜 이야기**를 담고 싶은 분

### 왜 이 서비스가 다른가요?

일반 AI 콘텐츠 생성과는 다릅니다.

**사장님이 직접 들려주신 이야기**를 바탕으로 콘텐츠를 만들어요.
짧은 인터뷰를 통해 고객 에피소드, 서비스 차별점, 실제 반응 같은
**진짜 경험**을 끌어내고, 이걸 4가지 형태의 콘텐츠로 변환합니다.

그래서 AI가 임의로 만들어낸 느낌이 아니라,
**"이건 우리 가게 이야기네"** 라고 느껴지는 콘텐츠가 나옵니다.

### 이렇게 진행돼요

```
STEP 1. 사진/영상 보내기
  → 텔레그램으로 가게 사진 5장 + 영상(있으면) 전송
  → 5분이면 충분합니다

STEP 2. 짧은 인터뷰
  → 텔레그램에서 3가지 질문에 음성 또는 텍스트로 답변
  → 사진을 기반으로 맞춤 질문이 자동 생성됩니다
  → "이 날 어떤 상황이었고 어떻게 이루어진 건지 들려주세요"

STEP 3. 콘텐츠 생성
  → 사장님의 답변을 바탕으로 4가지 콘텐츠를 제작합니다

STEP 4. 확인 및 수정
  → 미리보기 링크를 보내드립니다
  → 수정 요청 → 반영 → 최종 승인

STEP 5. 전달
  → 승인 후 모든 콘텐츠 파일 전달
```

### 제공 산출물 (4가지)

#### 1. 네이버 블로그 글
- 사장님 시점의 서사형 블로그
- 실제 인터뷰 멘트 인용 활용
- 사진 적절한 위치 배치
- 해시태그 포함
- HTML 형식 (바로 발행 가능)

#### 2. 인스타그램 릴스 (숏폼 영상)
- 20~30초 세로 영상 (1080x1920)
- AI 내레이션 (자연스러운 한국어 TTS)
- 텍스트 오버레이 + 자막
- 사진 Ken Burns 모션 + 영상 편집
- 바로 업로드 가능한 MP4

#### 3. 인스타그램 캐러셀 (피드 게시물)
- 5장 슬라이드 (1080x1350)
- 사진 편집 + 텍스트 오버레이 디자인
- 하나의 서사로 연결되는 구성
- 캡션 + 해시태그 포함

#### 4. 쓰레드/SNS 텍스트
- 블로그 기반 숏폼 텍스트
- 바로 복사해서 올릴 수 있는 형태

## FAQ

**Q. 어떤 업종에 적합한가요?**
뷰티샵, 카페, 식당, 네일샵, 숙소 등 오프라인 매장을 운영하시는 모든 업종에 적합합니다.

**Q. 사진이 전문가가 찍은 사진이 아니어도 되나요?**
네, 스마트폰으로 찍은 일상 사진이면 충분합니다. 영상도 카카오톡 촬영 영상이면 됩니다.

**Q. 인터뷰는 어떻게 진행되나요?**
텔레그램 메신저로 진행됩니다. 사진을 보고 자동으로 맞춤 질문이 생성되며, 음성 또는 텍스트로 편하게 답변하시면 됩니다. 약 5분 정도 소요됩니다.

**Q. 블로그를 직접 올려주시나요?**
기본 서비스는 콘텐츠 파일 전달입니다. 네이버 블로그 직접 발행은 별도 옵션으로 제공됩니다.

**Q. 정기 이용이 가능한가요?**
네, 주간/월간 정기 구독 서비스도 운영 예정입니다. 별도 문의 주세요.
